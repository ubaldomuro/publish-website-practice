Elementos basicos de HTML

En la siguiente tabla se muestran los elementos basicos del HTML

   Etiqueta       Nombre    Descripción
<html> HTML Le dice al navegador que lo que está dentro de estas etiquetas es código html.
<head> Head La etiqueta head contiene la metadata (info acerca de la página que no se muestra) para una página web.
<title> Title Contiene el nombre de la página que se va mostrar en la pestaña del navegador.
<body> Body Es el cuerpo de la pagina, la parte principal.
<div> Division Divisiones que se usan para agrupar elementos HTML y dividir partes de la pagina.
<h1> Head o Encabezado Es un encabezado que se utiliza para los titulos.
<p> Parrafo Se usa para indicar parrafos de texto.
<span> Span Selecciona una pequeña parte del texto y le da un formato diferente.
<em> Enfasis hace énfasis en un texto, en general el navegador lo muestra como cursiva.
<strong> Resaltar Resalta texto importante, en general el navegador lo muestra como negritas.
<br> Break El texto pasa al siguiente reglon, no requiere closing tag.
<ul> Unordered list Muestra elementos en formato de lista, pero sin un consecutivo.
<ol> Ordered list Crea una lista con elementos en orden numérico.
<li> List item Elemento dentro de una lista.
<a> Anchor Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.
<button> Button Es un boton.
<script> Script Lo que se encuentre dentro de esta tag es javaScript
<img> Images Agrega imágenes, no requiere de etiqueta de cerrado, pero se agrega la diagonal antes del >.
<video> Video Agrega un archivo de video.
<table> Table Crea una tabla.
<thead> Table head Secciona las casillas que seran encabezados.
<tbody> Table body Datos generales de la tabla.
<tfoot> Table foot Parte final de la tabla, se usa para los resultados, como por ejemplo una suma.
<tr> Table row Crea una fila en la tabla.
<th> Table heading El encabezado de una fila o columna.
<td> Table data Agrega una celda con datos en la tabla.
